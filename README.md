# Quick Crave

Quick Crave is a full-stack food delivery platform that connects customers with local restaurants. It features an intuitive frontend for seamless menu browsing and order placement, a robust backend for secure payments, and a dedicated admin dashboard for efficient restaurant management. Enjoy a fast, reliable, and user-friendly dining experience.

## Project Structure
- **/frontend**: User-facing food delivery application.
- **/admin**: Admin dashboard for restaurant and order management.
- **/server**: Node.js/Express backend API.

## Getting Started

1. **Install dependencies**
   Navigate into each folder and run `npm install`:
   ```bash
   cd frontend && npm install
   cd ../admin && npm install
   cd ../server && npm install
   ```

2. **Run the local development servers**
   Start the frontend, admin, and backend server respectively by following their specific package.json scripts (e.g. `npm run dev` or `npm start`).
